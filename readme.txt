This plugin allow WordPress to detect Real visitors IP Address.

== Description ==

This plugin allow WordPress to detect Real visitors IP Address when WordPress is behind of Reverse Proxy, Load Balancer or CDN.
It will start working as soon as you activate it. 

It is also compatible:

*  Cloudflare.com
*  geniusguard.com
*  central-hosting.com
*  incapsula.com
*  sucuri.net
*  barracuda.com
*  f5.com And Other...



== Installation ==

1. Go to Plugin Upload page ,http://example.com/wp-admin/plugin-install.php?tab=upload
1. Select "Real-IP-detector.zip" from Your Pc And install it.
1. It will start working as soon as you activate it.
